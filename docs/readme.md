# handlebarsjs 学习记录

> handlebars 使用记录

`Handlebars` 是 `JavaScript` 一个语义模板库，通过对`view` 和 `data` 的分离来快速构建Web模板。它采用 "Logic-less template" （无逻辑模版）的思路，在加载时被预编译，而不是到了客户端执行到代码时再去编译， 这样可以保证模板加载和运行的速度。`Handlebars` 兼容 `Mustache`，你可以在 `Handlebars`中导入 `Mustache` 模板。
